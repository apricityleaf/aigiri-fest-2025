# aigiri-fest-2025
文化祭で使用したサイネージ用データのうち、機密データのないものを公開

## Osakana Monitor
カメラから顔を認識して、口や目の動きに合わせて画面上の魚が動くサイネージ  
廊下に設置することで、来場者の企画までの待ち時間を退屈させないようにしました。  
[GitHub Pages](https://apricityleaf.github.io/aigiri-fest-2025/osakana-monitor/) から実際に確認できます。(カメラをONにしてください。)

## Timer
室内イベントで用いた競技用のタイマー  
初期値は50秒×2回戦ですが適当にパラメータを弄れば色々な用途で使えます。  
[GitHub Pages](https://apricityleaf.github.io/aigiri-fest-2025/timer/) から実際に確認できます。
